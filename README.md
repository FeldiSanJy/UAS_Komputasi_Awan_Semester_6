# 🚀 Company Profile: Percetakan Karya Mitra

Website profil perusahaan untuk Percetakan Karya Mitra. Proyek ini dibuat untuk memberikan informasi layanan percetakan secara profesional kepada pelanggan, sekaligus menyediakan panel pengelolaan data untuk administrator.

### 🌟 Fitur Utama

**Sisi Pengunjung (User)**
- Beranda: Informasi pembuka dan daya tarik utama perusahaan.
- Tentang Kami: Profil perusahaan.
- Layanan: Detail jenis layanan percetakan yang ditawarkan.
- FAQ: Daftar pertanyaan yang sering diajukan pelanggan.
- Kontak: Halaman untuk memudahkan pelanggan menghubungi perusahaan.

**Sisi Admin (Panel Pengelola)**
- Kelola Produk: Menambah, mengubah, atau menghapus daftar produk percetakan.
- Kelola Kategori: Mengatur pengelompokan produk agar lebih rapi.
- Backup & Restore: Fitur keamanan untuk mencadangkan dan memulihkan data database.

### 🛠️ Teknologi
- Front-end: HTML5, CSS3, JavaScript.
- Back-end: PHP & MySQL (Database diolah melalui Laragon).

### ⚙️ Cara Menjalankan
1. Pastikan Laragon aktif (Apache & MySQL).
2. Letakkan folder proyek di C:\laragon\www\.
3. Import database melalui phpMyAdmin.
4. Atur koneksi database di admin/db_config.php.
5. Buka di browser: http://localhost/PercetakanKaryaMitra/.
